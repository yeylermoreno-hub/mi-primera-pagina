## Temas vistos en clase

### ¿Qué es un programa?
Un programa es un conjunto de **instrucciones** que una computadora puede entender y ejecutar. Sirve para automatizar tareas, resolver problemas o crear aplicaciones. Por ejemplo, cuando escribimos código en Python, estamos dando órdenes paso a paso que el computador sigue.

### ¿Qué es una variable?
Una **variable** es un espacio en la memoria del computador donde se guarda un valor. Es como una “caja” con un nombre, que podemos usar para almacenar información y cambiarla cuando queramos.
